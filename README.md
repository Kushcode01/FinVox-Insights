# FinVox-Insights
This is repository of a POC voice based financial data visualization tool 

We can visualize the following financial data of any publically listed company in NSE,

- Earnings
- Balance Sheet
- Stock Prices
- Indicators and ratios

 in form of 
- Bar Plots
- Candle stick charts
- Line plots
- 3-D plots
